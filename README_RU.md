# Core5 Custom Link System

Это своя система ссылок для твоего личного Core5 Mod.

Она делает ссылки вида:

```text
https://твой-домен.ru/l/AbC12345
```

По ссылке открывается твоя страница, там показывается ID шара/аккаунта и считается статистика переходов.

Важно: система **не входит в чужой аккаунт** и **не обходит авторизацию игры**. Если сама игра требует вход через официальный способ, игрок всё равно должен войти сам. Эта система нужна для своих ссылок, красивого входа через твой домен и отслеживания переходов.

## Быстрый запуск на компьютере

1. Установи Node.js 18 или новее.
2. Открой папку с этим проектом.
3. Запусти:

```bash
npm start
```

4. Открой:

```text
http://localhost:3000
```

5. В моде укажи сервер ссылок:

```text
http://localhost:3000
```

## Настройки

Создай переменные окружения:

```text
PORT=3000
BASE_URL=https://твой-домен.ru
GAME_URL=
```

`BASE_URL` — твой домен для ссылок.

`GAME_URL` — официальный публичный адрес входа в игру, если он у тебя есть. Если такого адреса нет, оставь пустым. Тогда страница будет просто показывать ID и кнопку копирования.

Не ставь сюда `/vkontakte/`, `/odnoklassniki/`, `viewer_id`, `auth_key`, `session`, `token` и похожие параметры.

## API

Создать ссылку:

```bash
curl -X POST https://твой-домен.ru/api/links \
  -H "Content-Type: application/json" \
  -d '{"ownerId":"208","label":"Мой шар"}'
```

Ответ:

```json
{
  "ok": true,
  "url": "https://твой-домен.ru/l/AbC12345"
}
```

Статистика:

```text
https://твой-домен.ru/api/links/AbC12345/stats
```

## Куда залить

Подойдёт любой Node.js хостинг: VPS, Render, Railway, Replit, Glitch, свой сервер.

На хостинге обязательно укажи `BASE_URL` равным твоему настоящему домену.

## Файлы данных

- `data/links.json` — созданные ссылки.
- `data/clicks.json` — переходы по ссылкам.

Для маленького личного мода этого достаточно. Если переходов будет много, лучше заменить JSON на SQLite/PostgreSQL.
